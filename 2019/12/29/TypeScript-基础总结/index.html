# TS 的缺点？

有一定学习成本，需要理解接口、泛型、类、枚举等前端开发可能不是很熟悉的知识点

# TS 中的接口？

接口就是一个规范或者约束，让我们的对象都去遵循它

# 什么是 TS？

TS就像是法律它是针对任何人的，我们在做某一件事时就要遵循相关的约束

# 泛型？

泛型就是在定义函数或者接口的时候，不预先指定具体类型，而在使用的时候再指定类型的一种特性。
