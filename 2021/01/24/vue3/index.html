# composition api - 组合式 api

# setup

  • 参数：

    props：因为 props 是响应式的，不可以解构，如果解构将失去响应式，如果需要解构 prop，可以通过使用 setup 函数中的 toRefs 来完成此操作

    context：是一个普通的 js 对象，{attrs, slots, emit}，所以可以去解构

  • 访问组件的属性

    执行 setup 时，组件实例尚未被创建。因此，你只能访问以下 property：

    • props
    • attrs
    • slots
    • emit

    换句话说，你将无法访问以下组件选项：

    • data
    • computed
    • methods

  • 结合模板使用

    返回的对象，可以在 template 里直接使用

  • this：没有 this
