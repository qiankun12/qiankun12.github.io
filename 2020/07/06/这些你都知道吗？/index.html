# 前端页面构成

首先要知道W3C标准是什么？
    W3C标准万维bai网联盟du标准。
    万维网联盟（外语缩写：baiW3C）标准不du是某一个标准，而是zhi一系列标准的集dao合。网页主要由三部分组成：结构（Structure）、表现（Presentation）和行为（Behavior）。
    对应的标准也分三方面：结构化标准语言主要包括XHTML和XML，表现标准语言主要包括CSS，行为标准主要包括对象模型（如W3C DOM）、ECMAScript等。这些标准大部分由W3C起草和发布，也有一些是其他标准组织制订的标准，比如ECMA（European Computer Manufacturers Association）的ECMAScript标准。
# 结构 表现 行为
结构（structure）:
    由 HTML 或 XHTML 之类的标记语言负责创建。标签，也就是那些出现在尖括号里的单词，对网页内容的语义含义做出了描述，但这些标签不包含任何关于如何显示有关内容的信息。
    如果比喻是一个人的话，就相当于是这个人的骨骼。
表现（Presentation）:
    由 CSS 负责创建。操作样式，也就是页面的布局与展示效果。相当于人的衣服。
行为（behavior）:
    负责回答“内容应该如何对事件做出反应”这一问题。这是 Javascript 语言和 DOM 主宰的领域。相当于这个人要干什么。

一个元素在各种盒模型下的算法：
标准:width+padding+border+margin;
IE (怪异盒模型)：width+padding

JavaScript 三部分:
ECMA（European Computer Manufacturers Association）
DOM（document object model）
BOM（browser object model）

# 数据类型
基本数据类型有：number（数字类型）string（字符串类型）boolean（布尔型）null（空，对象的占位符）undefined（未定义）
引用数据类型：array（数组）function（函数）obj（对象）

有时候很有用的 css 语法：
cursor: not-allowed; 鼠标不能点击样式
cursor: pointer; 鼠标小抓手样式
！important（css 样式后加） 表示此项很重要覆盖之前样式
white-space: nowrap; css 样式横向排布（不换行）
transform: translate (X,Y); 元素自身的中心 定位后调整位置常用
outline: none; input、button 输入框无隐式边框
box-shadow: 1px 1px 5px rgba (0, 0, 0, 0.2); 边框线阴影
text-decoration: none; 取消 a 标签下划线

元素背景图引用
background-image: url();
background-repeat: no-repeat;   
background-position: center;

transfrom:  translate\X\Y(50px)          平移
            scale\X\Y(一个值 或两个值)   缩放多少倍 可用于操作字体小于12px 例scale(0.65,0.65);
            rotate\X\Y(30deg)            旋转30度
            skew\X\Y(一个值 或两个值deg) 倾斜**度
